Beta-VAE used for clustering 1¶

Type of Variational Auto-Encoder where the KL divergence term in the loss function is weighted by a parameter beta.

Training attributes: bleeding, plt, shock, haematocrit_percent, bleeding_gum, abdominal_pain, ascites, bleeding_mucosal, bleeding_skin, body_temperature.

Attributes used in cluster comparison: age, gender, weight.
